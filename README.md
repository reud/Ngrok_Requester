Ngrok Requester
====

大学の授業の自由課題で作成した、ngrokのURLにGETリクエストをほんのちょこっとだけ簡単に送れるアプリケーション

## Description
ngrokのURLにGETリクエストにワンクリックでGETリクエストを送ります([文字列].ngrok.io/のみ)
